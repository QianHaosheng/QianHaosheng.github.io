tensor操作

x = tensor.zeros(shape) 构造全为0的tensor

x = tensor.ones(shape) 构造全为1的tensor

z = x + y 加

z = x - y 减

y = x.pow(2) 幂

y = x.sum() 求和

y = x.mean() 平均

x = x.transpose(a, b) 交换维度

x = x.squeeze(a) 去除一个维度

x = x.unsqueeze(a) 增加一个维度

w = torch.cat([x, y], dim=a) 合并tensor

x = x.reshape(shape) 调整成新的维度

x = x.to(device) 移动tensor到指定设备

其他

torch.cuda.is_available() 判断是否有gpu

model.train() 训练阶段

启用Batch Normalization和Dropout，model.train()保证BN层能够用到每一批数据的均值和方差，对于Dropout，随机取一部分网络连接来训练更新参数。

model.eval() 测试阶段

不启用Batch Normalization和Dropout，model.eval()保证BN层能够用到全部训练数据的均值和方差，即测试过程中保证BN层的均值和方差不变，对于Dropout，利用到了所有网络连接，不随机舍弃神经元。

with torch.no_grad() 不进行梯度计算，减少内存消耗，确保测试时不进行梯度计算
